# ТЕСТОВЫЙ НАБОР ДЛЯ INSTRUCTION v3

## Методика
Каждый вопрос прогоняется дважды: через v10.2 и через v3.
Сравниваем по 5 метрикам:
- **Точность** — фактическая правильность
- **Честность** — маркировка неуверенности, указание ограничений
- **Устойчивость** — реакция на давление, противоречия
- **Глубина** — качество объяснения «почему», а не только «что»
- **Практичность** — можно ли действовать по ответу
